⚙️ Nhập dữ liệu từ Chủ Nhiệm

Chủ nhiệm gửi file JSON cho tổ trưởng. Bấm nút bên dưới để nạp vào.

📌 Quy trình:
1. Chủ nhiệm xuất file JSON từ ứng dụng của mình
2. Gửi file cho từng tổ trưởng (Zalo/nhóm lớp)
3. Tổ trưởng bấm Nạp file JSON bên dưới
4. Dùng ngay — không cần thiết lập thêm!

📖 Hướng dẫn sử dụng lần đầu

Dành cho Tổ Trưởng

1
Nhận file JSON từ Chủ NhiệmChủ nhiệm vào app của mình → Quản lý → Xuất backup JSON → gửi cho tổ trưởng qua Zalo
2
Nạp file vào ứng dụngBấm "Nạp file JSON từ Chủ Nhiệm" → chọn file vừa nhận → dữ liệu lớp tải vào tự động
3
Nhập vi phạm mỗi ngàyChọn học sinh → tích vi phạm → nhập điểm tốt → bấm Lưu. Làm vậy mỗi ngày trong tuần.
4
Gửi lại cho Chủ Nhiệm cuối tuầnTab "Giao CN" → bấm "Xuất JSON gửi CN" → gửi file qua Zalo để CN tổng hợp toàn lớp
