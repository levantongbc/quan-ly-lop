📊 Tổng kết lớp – Tuần

📥 Nhận file từ Tổ Trưởng

Cuối tuần, tổ trưởng gửi file JSON qua Zalo. Bấm nạp từng file để gộp dữ liệu vào hệ thống.

⚠️ Nạp file của tổ nào thì dữ liệu tổ đó được cập nhật. Các tổ khác giữ nguyên.

📤 Gửi file cho Tổ Trưởng

Đầu năm hoặc khi cập nhật danh sách — xuất file và gửi cho từng tổ trưởng.

📧 Gửi email cuối tuần

⚠️ Chỉ gửi vào Thứ Sáu hoặc Thứ Bảy. Email gửi đến phụ huynh toàn lớp.

📋 Lấy Apps Script

Dán code này vào Google Sheets → Apps Script → Deploy as Web App → lấy URL.

✏️ Thiết lập lại danh sách học sinh

Chỉnh sửa tên lớp, giáo viên hoặc cập nhật danh sách học sinh. Điểm & vi phạm hiện tại giữ nguyên.

🔄 Chuyển tổ học sinh

Chuyển học sinh giữa tổ, lịch sử điểm giữ nguyên.
